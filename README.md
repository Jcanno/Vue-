# vue-dianping 仿大众点评Vue项目

## 技术栈:
* Vue
* Vuex
* Vue-Router
* Vue-Cli
* express
* sequelize
* postgreSql


## 实现功能:
* 登录注册
* 发布点评
* 点赞评论
* 上传文件


## 下载并安装模块
```
git clone https://github.com/Jcanno/Vue-DianPing.git
cd Vue-DianPing
npm install
```

## 运行编译程序
```
npm run serve
```

## 编译程序发布
```
npm run build
```

### 示例所用node.js的orm框架sequelize,数据库postgreSql,数据库配置在'/server/config/dbConfig'中
